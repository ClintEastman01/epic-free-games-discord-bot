Purpose: Configure the bot for your server

üÜò Support & Troubleshooting

Common Issues

Bot not responding to commands

Make sure the bot has been configured with /setup and you're using commands in the configured channel.

Permission errors

Ensure the bot has "Send Messages" and "Use Slash Commands" permissions in your server.

No game notifications

The bot only sends notifications for new games. Use /games to see current games.
